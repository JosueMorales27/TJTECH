Módulo de Videojuegos para Flipper Zero

Mejora tu Flipper Zero con nuevas funciones de juego y desarrollo.

Módulo de Videojuegos

$49.00

El Módulo de Videojuegos para Flipper Zero, alimentado por un microcontrolador Raspberry Pi RP2040, te permite jugar en tu TV, utilizar un sensor de movimiento integrado para experiencias interactivas y conectar componentes DIY a través de un puerto GPIO de 14 pines. También cuenta con un puerto USB-C para comunicación directa con PC y actualizaciones de firmware, y puede funcionar como un dispositivo independiente.

  • Microcontrolador Raspberry Pi RP2040
  • Salida de video para pantalla de TV
  • Sensor de movimiento con API abierta
  • Puerto GPIO de 14 pines para proyectos DIY
  • Puerto USB-C para comunicación con PC
  • Modo independiente con firmware alternativo
